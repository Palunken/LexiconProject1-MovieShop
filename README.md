# 🎬 Movie Shop

## 📝 Description

Movie Shop is a group project developed during Lexicon's System Development program in 2025.

The goal of the project was to build an online movie store where users can browse, purchase, and manage movie orders. The project was developed over a ten-day period using ASP.NET Core MVC, Entity Framework Core, and SQL Server.

---

## ✨ Features

* Browse the movie catalog
* View detailed movie information
* Add movies to the shopping cart
* Place orders through a simple checkout process
* Customer information is saved and reused for future orders
* Customer and order management pages
* Pagination for the movie catalog

# 🚀 Running the Project

## Prerequisites

* Visual Studio 2026 (tested)
* .NET 8 SDK

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Palunken/MovieShop.git
```

Alternatively, download the repository as a ZIP file from GitHub.

2. Open the solution (`MovieShop.sln`) in Visual Studio.

3. Restore the NuGet packages if prompted.

4. Press **F5** or click **Start Debugging** to build and run the application.

No additional database setup is required. The application automatically creates and populates the database with sample movie data on first run.

---

## 🛒 Using the Application

* Browse the movie catalog.
* Add movies to the shopping cart.
* Proceed to checkout.
* When placing an order for the first time, enter your customer information to create a customer profile.
* For future orders, simply enter the same email address to reuse your existing customer information.
